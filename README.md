DARPA INFORMÁTICA

Link para site ja hospedado: https://darpa-lorenzo.netlify.app/

Esse website foi desenvolvido para aprendizado das linguagens HTML e CSS.
Esse website simula uma empresa denominada DARPA Informática.
Com esse site foram ensinados conceitos do HTML como listas, formulários, tabelas, videos e imagens, html básico, mapas, tags com semántica.

Em css foram ensinados conceitos basicos como tamanho e cor de fontes, tamanho de altura, transições, hover, fontes internas e externas, espaçamento interno e externo.
